# 🌿 TO ME
깔끔하고 직관적인 플랜테리어 쇼핑몰 프로젝트


## 특징
- **반응형 페이지** : min-width 768px ~ 1920px 이상


### 파일 구조
```
├ images              이미지 파일
├ css                 css 파일
│
├ index.js          
├ App.js            
│
├ Mainpage.js         메인 페이지
├ Loginpage.js        서브 페이지: 로그인
├ Signuppage.js       서브 페이지: 회원가입
│
├ ProductsIndex.js    상품 페이지: 상위 라우터
├ Products.js         상품 페이지: 전체 상품
├ Plants.js           상품 페이지: 카테고리1 상품
├ Interior.js         상품 페이지: 카테고리2 상품
├ Tools.js            상품 페이지: 카테고리3 상품

```

## 기술 스택
- html5 : 시맨틱 마크업
- css3 : position, flex, animate, transition
